Quick Start

Last updated April 2026 · 5 min read

Installation

Get OpenClaw running on your server in under two minutes. The installer handles dependencies, configuration, and your first agent automatically.

# Install OpenClaw globally
npm install -g openclaw

# Initialize a new instance
openclaw init

# Start the gateway
openclaw gateway start
Note

Requires Node.js 20+ and a Linux/macOS environment. Windows support via WSL2.

Connect a Channel

OpenClaw communicates through channels — Telegram, Discord, WhatsApp, and more. Connect your first channel to start talking to your agent.

# Add Telegram as a channel
openclaw channel add telegram \
  --token "your-bot-token"

Once connected, send any message to your bot. The agent will respond using the default model configured in config.yaml.

Install Your First Skill

Skills extend your agent's capabilities. Browse ClawHub for community-built skills, or create your own.

# Search for weather skills
clawhub search weather

# Install a skill
clawhub install weather
