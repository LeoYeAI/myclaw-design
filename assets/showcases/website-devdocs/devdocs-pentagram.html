Installation

Get OpenClaw running on your machine in under 5 minutes. Supports Linux, macOS, and Docker.

Prerequisites

OpenClaw requires Node.js v20+ and npm v10+. We recommend using the latest LTS release.

● Important
OpenClaw runs as a system service. You'll need sudo access for the initial setup, but the agent runs as an unprivileged user.

Quick Install

The fastest way to get started is our one-line installer:

BashCopy
# Install OpenClaw globally curl -fsSL https://get.openclaw.ai | bash # Verify installation openclaw --version # → openclaw v4.2.0 # Start the gateway openclaw gateway start

Manual Installation

If you prefer manual control, install via npm:

BashCopy
npm install -g openclaw # Initialize workspace openclaw init # Configure your first channel openclaw channel add telegram \ --token "YOUR_BOT_TOKEN" # Set your AI model openclaw config set model "claude-opus-4"

REST API

OpenClaw exposes a REST API for programmatic control:

EndpointDescription
GET/api/statusGateway and agent status
GET/api/skillsList installed skills
POST/api/skills/installInstall a skill from marketplace
POST/api/messageSend a message to the agent
GET/api/memoryRead agent memory files
PUT/api/configUpdate configuration
DELETE/api/skills/:idUninstall a skill

Docker

DockerCopy
docker run -d \ --name openclaw \ -p 3000:3000 \ -v openclaw-data:/home/ubuntu/.openclaw \ -e ANTHROPIC_API_KEY=sk-ant-... \ -e OPENCLAW_MODEL=claude-opus-4 \ openclaw/openclaw:latest
