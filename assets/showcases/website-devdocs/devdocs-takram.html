Welcome to OpenClaw

OpenClaw is the open-source AI agent platform that gives you a full personal server with complete code control. Build, deploy, and manage autonomous AI agents.

💡
New to OpenClaw? Start with the Quick Start guide to get your first agent running in under 5 minutes. No prior AI experience needed.

Installation

Install OpenClaw globally using npm. This gives you the openclaw CLI and the gateway daemon.

bash 📋 Copy
# Install OpenClaw globally npm install -g openclaw # Initialize your workspace openclaw init # Start the gateway openclaw gateway start # Check status openclaw status ● Gateway running on :3000 ● Agent ready ● Skills 0 installed

Core Architecture

OpenClaw follows a modular architecture with three main components: the Gateway (HTTP/WebSocket server), the Agent runtime (LLM orchestration), and the Skill system (extensible capabilities).

javascript 📋 Copy
// Example: Register a custom skill import { defineSkill } from 'openclaw/sdk'; export default defineSkill({ name: 'my-skill', description: 'A custom skill', triggers: ['my command'], async execute(ctx) { return ctx.reply('Hello from my skill!'); } });
